# Allan White Photo

A personal photography publishing platform built with Astro, Sanity, Cloudflare Workers, and Cloudinary for video clips.

## Commands

```sh
pnpm install
pnpm build
pnpm dev:web
pnpm dev:studio
```

## Environment

Copy `.env.example` to `.env` and fill in the Sanity values before connecting real content. The web app builds without those values and shows empty states.
